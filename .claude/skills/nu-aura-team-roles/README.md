# NU-AURA Team Role Skills

This directory contains role-specific skill documents for the NU-AURA platform development team.

## Available Role Skills

### Leadership (2 roles)
1. **[Engineering Manager / Tech Lead](01-engineering-manager-tech-lead.md)** - Sprint planning, architectural consistency, code quality, team leadership
2. **[Product Manager](02-product-manager.md)** - Roadmap ownership, requirements definition, customer discovery, launch strategy

### Backend Engineering (4 roles)
3. **[Senior Backend Engineer - Core Platform](03-senior-backend-core-platform.md)** - Multi-tenant RLS, RBAC, approval engine, Kafka pipeline, scheduled jobs
4. **[Backend Engineer - NU-HRMS + Payroll](04-backend-hrms-payroll.md)** - Payroll engine, SpEL formulas, DAG evaluation, leave accrual
5. **[Backend Engineer - NU-Hire + NU-Grow](05-backend-hire-grow.md)** - Recruitment pipeline, onboarding, performance reviews, OKRs, LMS
6. **[Backend Engineer - NU-Fluence](06-backend-fluence.md)** - Elasticsearch integration, wiki/blog management, Drive integration

### Frontend Engineering (3 roles)
7. **[Senior Frontend Engineer - Platform](07-senior-frontend-platform.md)** - Design system, app switcher, auth flows, Mantine components, React Query patterns
8. **[Frontend Engineer - NU-HRMS](08-frontend-hrms.md)** - Employee management, attendance, leave, payroll, benefits, assets
9. **[Frontend Engineer - NU-Hire + NU-Grow](09-frontend-hire-grow.md)** - ATS UI, onboarding, performance, OKRs, surveys

### DevOps & Infrastructure (1 role)
10. **[DevOps Engineer](10-devops-engineer.md)** - Docker, Kubernetes, Prometheus, CI/CD, MinIO, Kafka cluster

### Quality Assurance (1 role)
11. **[QA Engineer](11-qa-engineer.md)** - E2E testing, tenant isolation validation, RBAC testing, automation

### Design (1 role)
12. **[UI/UX Designer](12-uiux-designer.md)** - Design system, mockups, B2B SaaS patterns, consistency across 4 sub-apps

### Domain Expertise (1 role)
13. **[HR Domain Expert](13-hr-domain-expert.md)** - Payroll compliance, leave policies, statutory requirements (India/global)

## Total Team Size

**Core Team**: 8-10 members
- Leadership: 2
- Backend: 4
- Frontend: 3
- DevOps: 1
- QA: 1
- Design: 1

**Extended Team**: +1 HR Domain Expert (part-time/consulting)
