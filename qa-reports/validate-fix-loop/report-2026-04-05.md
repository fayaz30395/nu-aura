### Pass 3 Errors (verified after server restart + hard refresh)

**ZERO 403 console errors for EMPLOYEE role.** Fix approach: `validateStatus: (s) => s < 500` in
axios config prevents 403 from being treated as an error, so the apiClient interceptor never logs
it. Services check `response.status === 403` and return empty gracefully.

Only remaining errors are **backend timeouts** (P3 — transient, caused by load):

- GET /announcements/active — timeout 30s
- GET /recognition/feed — timeout 30s
- GET /home/anniversaries — timeout 30s
- GET /home/new-joinees — timeout 30s
