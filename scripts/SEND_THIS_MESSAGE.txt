TROUBLESHOOTING QUICK FIXES
================================================================================

Issue: Container won't start
Fix:  docker-compose logs postgres
      docker-compose restart postgres

Issue: Port already in use
Fix:  sudo lsof -i :5432
      # Kill the process or change port in docker-compose.yml

Issue: Database already exists
Fix:  docker exec hrms-postgres psql -U postgres -c "DROP DATABASE IF EXISTS hrms_dev;"
      # Then re-import

Issue: Permission denied
Fix:  docker exec hrms-postgres psql -U postgres -c "GRANT ALL PRIVILEGES ON DATABASE hrms_dev TO hrms;"

Issue: Services unhealthy
Fix:  docker-compose down
      docker-compose up -d
      docker-compose ps
