## Security Architecture (Verified)

All major security patterns implemented correctly:

- **JWT:** Stateless, 1-hour expiry, 24-hour refresh
- **RBAC:** Roles in JWT, permissions from DB per request
- **SuperAdmin:** Bypasses all access control
- **Tenant Isolation:** TenantFilter + PostgreSQL RLS
- **Rate Limiting:** 5/min auth, 100/min API, 5/5min export
- **CORS:** localhost:3000/3001/8080, no wildcards
- **CSRF:** Double-submit cookie pattern
- **Headers:** HSTS, CSP, X-Content-Type-Options, Referrer-Policy, Permissions-Policy
